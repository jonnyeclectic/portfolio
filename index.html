Open Source

boost — a package manager for what your agent knows

An independent, open-source CLI that installs and governs AI agent skills. Built and maintained on my own time.

boost pulls AI coding skills out of Git-hosted registries and lands them in Claude Code, Windsurf, and Cursor in one step — 72 commands spanning install, discovery, AI-assisted authoring, governance, and team rollout.

Skills get handled like real dependencies instead of files you paste around: a lock file pins exact state, boost audit screens against safety blocklists, and Boostfile bundles hand an entire setup to someone else intact.
